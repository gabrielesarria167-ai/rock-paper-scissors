This is my first JavaScript project, where my goal will be to design a very basic rock, paper, scissors game,that will be ran entirely from the console.

The game itself will be played against the computer, who will get its choice by using the _Math.random()_ function, and will include several rounds and an overall winner.

I will include comments on several lines of code for the people who are curious to see how the program works.

---

## OBJECTIVE AND PLANNING

The objective was already declared before: create a basic rock, paper, scissors game.

For this project I will need:
- A way to get the computer choice, which has to be randomized every turn .
- A way to grab and store the users choice, which will then be confronted against the computers' one by either using ifs or a switch condition.
- An indicator that shows the total rounds played and that have yet to be played.
- An indicator that shows the current score between the computer and the user.
- A final output that shows whether the user won the majority of games, the computer did, or if it was a draw.
